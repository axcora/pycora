# About PyCora

**PyCora** is a lightweight, fast, and elegant static site generator built with Python. It was created to provide developers and content creators with a simple yet powerful tool for building static websites.

A project present by [Axcora Technology](https://axcora.com)

---

## 🚀 What is PyCora?

PyCora is a **Static Site Generator (SSG)** that converts Markdown files with YAML frontmatter into fully functional HTML websites. It combines the simplicity of Markdown with the power of Python and Jinja2 templating.

### Key Features

- ⚡ **Lightning Fast** - Pure static HTML files
- 📝 **Markdown + YAML** - Write content in Markdown with frontmatter
- 🎨 **Beautiful Typography** - Clean, readable design
- 🔍 **SEO Optimized** - Meta tags, OG, Twitter Cards
- 📱 **Fully Responsive** - Bootstrap 5 framework
- 🏷️ **Tags & Categories** - Automatic organization
- 🔥 **Live Development** - Auto-rebuild and refresh
- 📡 **RSS & Sitemap** - Built-in generation

## 📖 How PyCora Works

1. **Write Content** - Create Markdown files in `content/posts/`
2. **Add Frontmatter** - Include YAML metadata (title, date, tags, etc.)
3. **Build** - Run `python ssg.py` to generate HTML
4. **Deploy** - Upload the `output/` directory to your hosting
